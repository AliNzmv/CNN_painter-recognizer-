# 🎨🖼️ Painter Recognition with CNN 🎉

## 📋 Project Overview  
This project is a **deep learning model** that uses **Convolutional Neural Networks (CNNs)** and **Transfer Learning** to predict the painter of a painting.  
The dataset was sourced from [quera.org](https://quera.org). 🌐  

---

## 🛠️ Project Features  
1. **Using ResNet50**  
   - The base model for this project is ResNet50, chosen for its robust feature extraction capabilities. 🔗  
2. **Transfer Learning**  
   - Leveraged pre-trained weights from ResNet50 to optimize training and improve efficiency. ⚡  
3. **Image Classification**  
   - The model classifies paintings to their respective painters. 🎨  

---



## 📊 Tools and Technologies  
- TensorFlow/Keras 🧠  
- ResNet50 🔗  
- Python 🐍  
- Matplotlib 📊  

---

## ✨ Project Highlights  
- **High speed** training using transfer learning.  
- **High accuracy** in painter classification with minimal data.  
- **Clean and modular design** for the codebase.
